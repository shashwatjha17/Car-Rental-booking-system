# UML Diagrams

This folder contains separate UML-style diagrams for the Car Rental full-stack project.

## Diagram Files

- [Class Diagram](./class-diagram.md): Mongoose domain models and relationships.
- [Use Case Diagram](./use-case-diagram.md): Customer, owner, and system capabilities.
- [Component Diagram](./component-diagram.md): Frontend, backend, database, and external service structure.
- [Booking Sequence Diagram](./booking-sequence-diagram.md): User booking creation flow.
- [Owner Booking Sequence Diagram](./owner-booking-sequence-diagram.md): Owner booking management and chat flow.
- [Activity Diagram](./activity-diagram.md): High-level car search and booking activity.
- [Deployment Diagram](./deployment-diagram.md): Runtime deployment view.

These diagrams use Mermaid syntax and can be previewed in Markdown viewers that support Mermaid.
